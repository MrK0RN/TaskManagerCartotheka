# Telegram-бот (PHP): уведомления о днях рождения и задачах

Версия бота на PHP, без Docker. Использует ту же БД PostgreSQL, что и основное приложение.

Бот присылает подписчикам:

**Дни рождения** (из портретов, параметр 1 — ФИО и дата рождения):
- **8:00 МСК** — у кого день рождения **сегодня**;
- **20:00 МСК** — у кого день рождения **завтра**.

**Задачи:**
- **8:00 МСК** — задачи на **сегодня**;
- **20:00 МСК** — задачи на **завтра**;
- **22:00 МСК** — напоминание внести обновление по задачам за сегодня, к которым прикреплён человек.

## Требования

- PHP 8.0+ с расширениями `pdo`, `pdo_pgsql`, `json`
- Доступ к PostgreSQL (та же БД, что и у сайта; таблица `telegram_subscribers`)

## Установка

1. Скопируйте конфиг и заполните:

   ```bash
   cp config.example.php config.local.php
   ```

2. В `config.local.php` укажите:
   - `TELEGRAM_BOT_TOKEN` — токен от @BotFather;
   - `SITE_URL` — публичный URL сайта (для ссылок в сообщениях);
   - `DB_HOST`, `DB_PORT`, `DB_NAME`, `DB_USER`, `DB_PASS` — подключение к PostgreSQL (как у основного приложения).

3. Убедитесь, что в БД есть таблица подписчиков (если сайт уже развёрнут по основной схеме — она уже есть). Иначе выполните:

   ```bash
   psql -U portrait_user -d portrait_db -f ../sql/telegram_subscribers_migration.pgsql
   ```

## Запуск бота (приём команд)

Пользователи пишут боту **/start** — чат добавляется в рассылку.

Команды по запросу:
- **/morning** — ДР и задачи на сегодня;
- **/evening** — ДР и задачи на завтра.

Запуск long-polling (держать процесс в фоне или через systemd/supervisor):

```bash
php poller.php
```

Один экземпляр бота на один токен. Если появится ошибка 409 Conflict — где-то уже запущен бот с этим токеном.

## Рассылка по расписанию (cron)

Рассылка в 8:00, 20:00 и 22:00 выполняется отдельными вызовами `notify.php` по cron. Пример (все времена — по Москве, настройте `TZ=Europe/Moscow` при необходимости):

```cron
# Дни рождения + задачи на день (одним вызовом утро/вечер)
0 8 * * * cd /path/to/telegram_bot_php && php notify.php --morning
0 20 * * * cd /path/to/telegram_bot_php && php notify.php --evening
# Напоминание по задачам с прикреплённым человеком
0 22 * * * cd /path/to/telegram_bot_php && php notify.php --tasks-remind
```

Либо раздельно (если нужно раскидать по разным минутам):

```cron
0 8 * * * cd /path/to/telegram_bot_php && php notify.php --tasks-morning
0 20 * * * cd /path/to/telegram_bot_php && php notify.php --tasks-evening
```

## Конфигурация

| Параметр | Описание |
|----------|----------|
| `TELEGRAM_BOT_TOKEN` | Токен от @BotFather (обязательно) |
| `SITE_URL` | Публичный URL сайта для ссылок в сообщениях |
| `DB_HOST`, `DB_PORT`, `DB_NAME`, `DB_USER`, `DB_PASS` | Подключение к PostgreSQL |

Значения можно задать в `config.local.php` или в переменных окружения.
